# AWS Cloud Quest

During the AWS re/Start programme I also worked through AWS Cloud Quest modules.  
It is a gamified learning environment where you solve small challenges and learn how AWS services work in practice.

I like this format because it forces you to actually do things instead of only reading theory.

## What I practiced

Working through the Cloud Quest scenarios helped me practice:

- basic cloud concepts
- launching and managing AWS resources
- understanding how different AWS services connect
- troubleshooting when something does not work

---

## AWS services used

During these exercises I worked with:

- Amazon EC2  
- Amazon S3  
- AWS IAM  
- Amazon VPC  
- AWS CloudWatch

## Personal note

What I find interesting is how many cloud topics connect to things I have seen before in business.

For example, when learning about IAM and security policies I immediately think about governance and control frameworks from finance.  
And when AWS talks about cost management, my finance brain automatically starts thinking about budgets and cost allocation.

For me this learning process is not only about technology.  
It is about understanding the infrastructure behind modern companies.
