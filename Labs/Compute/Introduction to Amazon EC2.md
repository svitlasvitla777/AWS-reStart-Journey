# Amazon EC2 – Hands-On Lab

## Overview
Completed a practical AWS lab focused on deploying and managing compute infrastructure
using Amazon EC2 — covering the full lifecycle of an instance from launch to termination.

## What I Did
Deployed and managed an Amazon EC2 web server instance, applying real-world configuration
and operational best practices in the AWS Management Console.
