# Amazon EC2 – Hands-On Lab

## Overview
Completed a practical AWS lab focused on deploying and managing compute infrastructure
using Amazon EC2 — covering the full lifecycle of an instance from launch to termination.

## What I Did
Deployed and managed an Amazon EC2 web server instance, applying real-world configuration
and operational best practices in the AWS Management Console.

## Skills Demonstrated
- Provisioned an EC2 instance with termination protection to prevent accidental deletion
- Monitored instance health and performance metrics via AWS CloudWatch
- Configured and modified Security Groups to control inbound HTTP traffic
- Scaled compute capacity by resizing the EC2 instance type — demonstrating elasticity
- Validated termination protection behaviour before executing a clean instance teardown

## Key Takeaways
Deepened understanding of AWS compute infrastructure — specifically how EC2 enables
rapid provisioning, flexible scaling, and cost-efficient resource management.
Reinforced knowledge of cloud security fundamentals through hands-on Security Group
configuration and access control.

## Tools & Services
- Amazon EC2
- AWS Management Console
- Security Groups
- Amazon CloudWatch
- AWS IAM (access context)

## Relevance
Applicable to roles involving cloud infrastructure, DevOps, or backend engineering
where AWS compute knowledge is expected.
